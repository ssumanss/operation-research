    The main method to solve linear programming problem in python is given by the following command

    scipy.optimize.linprog(c, 
                       A_ub=None, 
                       b_ub=None, 
                       A_eq=None, 
                       b_eq=None, 
                       bounds=None, 
                       method='revised simplex', 
                       callback=None, 
                       options=None, 
                       x0=None)

    Example 4¶

    Consider the following linear programming problem 

    c = [-3, -2]
A = [[4, -1],
     [4, 3],
     [4, 1]]
